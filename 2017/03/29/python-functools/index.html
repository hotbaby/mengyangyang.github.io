functools.wraps()

wraps(wrapped[,assigned][,updated])
常用装饰器函数中返回的wrapper()函数，解决了被装饰函数name、doc等签名丢失问题。

functools.partial()

partial(func[,*args][,**kwargs])
是一个装饰器函数，也是一个闭包，返回一个可调用对象，freeze一些参数。
