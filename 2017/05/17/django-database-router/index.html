django ORM数据模型配置数据库．

django支持多个数据库，通过django ORM定义数据模型，比如class User(Model)，无法通过class Meta配置管理该数据模型对应的数据库，只能使用默认数据库default．

django ConnectionRouter解决数据模型与数据库映射．

配置DB routers

DATABASE_ROUTERS = ['db_router.DatabaseRouter']
